# 🍕 Pizza Sales Intelligence Dashboard

## 📌 Overview

This project presents an **end-to-end data analytics solution** combining:

- 🗄️ **SQL (Data Analysis & Exploration)**
- 📊 **Power BI (Data Visualization)**
- 🧠 **Business Intelligence (Insights & Decision-Making)**

The goal is to transform raw pizza sales data into **meaningful insights** that help businesses improve performance, optimize operations, and drive revenue.

## 🎯 Project Objective

To analyze pizza sales data and:

- 📈 Evaluate **revenue performance & KPIs**
- 🍕 Identify **top-performing products**
- 📅 Understand **customer ordering behavior**
- 💡 Generate **data-driven business insights**

## 🧠 Dataset Overview

The dataset includes:

- Orders  
- Order Details  
- Pizza Types  
- Pizza Categories  
- Size variations  

Covering the period from **Jan 2015 to Dec 2015**.

---

## 🛠️ Role of SQL in This Project

SQL was used as the **foundation for data analysis** before visualization.

### 🔍 What Was Analyzed Using SQL

- 📊 **Sales Performance**
  - Total revenue, total orders, and average order value  
- 🍕 **Product Analysis**
  - Top-selling pizzas and category contribution  
- 📅 **Customer Behavior**
  - Orders by day and time patterns  
- 📈 **Trend Analysis**
  - Monthly performance and peak periods  
- 📦 **Size Analysis**
  - Quantity distribution across pizza sizes  

---

### 🎯 Why SQL Was Used

- ✔ To **extract and aggregate data efficiently** from multiple tables  
- ✔ To **join relational datasets** (orders, pizzas, categories)  
- ✔ To perform **accurate calculations** (revenue, averages, contributions)  
- ✔ To enable **deep data exploration before visualization**  
- ✔ To ensure **clean, structured data** for Power BI  

---

## 🧩 Dashboard Features

### 🔹 1. Sales Trend Analysis
- Monthly revenue and order trends  
- Identification of peak sales periods  

### 🔹 2. Product Performance
- Top-selling pizzas by revenue  
- Product-level comparison  

### 🔹 3. Category Analysis
- Revenue contribution by category:
  - Classic  
  - Supreme  
  - Chicken  
  - Veggie  

### 🔹 4. Size Analysis
- Quantity sold by pizza size  
- Customer preference insights  

### 🔹 5. Customer Behavior Analysis
- Orders by day of the week  
- Peak ordering patterns  

### 🔹 6. Interactive Dashboard
- Date range slicer  
- Category filter  
- Size filter  

---

## 📊 Key Insights

- 🔥 Sales peak on **weekends**, with Friday highest  
- 📈 **July is the highest revenue month**  
- 🍕 Large-size pizzas drive the majority of revenue  
- 🎯 A small group of products contributes significantly to total sales
